CRUD desenvolvido com Node.js

Endpoints

  • SYNC DATABASE: /database
    curl --location 'http://localhost:5000/database'
  • CREATE: /programmer
    curl --location 'http://localhost:5000/programmer' \ --header 'Content-Type: application/json' \ --data '{ "name": "João Pedro Santana Marques", "javascript": true, "java": true, "python": true }'
  • READ: /programmer
    curl --location --request GET 'http://localhost:5000/programmer/' \ --header 'Content-Type: text/plain' \ --data '{ "id": 1; }'
  • READ: /programmer/id
    curl --location 'http://localhost:5000/programmer/1' \ --data ''
  • UPDATE: PUT /programmer
    curl --location --request PUT 'http://localhost:5000/programmer' \ --header 'Content-Type: application/json' \ --data '{ "id": 1, "name": "João Pedro" }'
  • DELETE: /programmer
    curl --location --request DELETE 'http://localhost:5000/programmer/' \ --header 'Content-Type: application/json' \ --data '{ "id": 1 }'
  • DELETE: /programmer/id
    curl --location --request DELETE 'http://localhost:5000/programmer/1' \ --data ''
